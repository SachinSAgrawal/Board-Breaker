# BOARD BREAKER 

## INSTRUCTIONS
Two expert Tae-kwon-do fighters duel for an epic battle lasting one minute.

## GOAL
Break the most boards in one minute to become the Kung Fu Fighting Champion.

## CONTROLS
```
Player 1: Arrows
↓ - moves down
↑ - moves up
← - moves left
→ - moves right
Player 2: WASD
Key S - moves down
Key W - moves up
Key A - moves left
Key D - moves right
```
